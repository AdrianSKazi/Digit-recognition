# Digit-recognition

### task from Kaggle

Used approach:

* DNN
* 10 inner layers 1 with 556 dimensions and 9 with 128
* activation function: relu
* optimizer: adam
* loss: sparse categorical entropy

Outcome:
* Training model final accuracy: 98.94%
* Validation accuracy: 97.53%

Dataset: MNIST
